# sevux 2.0
> Mini jeu sur navigateur développé avec vanilla javascript, html et css
## Description
L'utilisateur contrôle un véhicule lui permettant de tirer sur des cibles qui arrivent sur lui provenant du haut de la zone de jeu.
> **Note**: Ceci est une  amélioration de la précédente version ([sevux](https://github.com/brelnosse/sevux)) dévéloppé il y a environ 4 ans de cela.
